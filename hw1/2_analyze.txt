Верхняя часть шапки сожержит ссылки на информацию о пользователе:
1. Состояние авторизации
2. Обсуждения с ID пользователя
3. Вклвды с ID пользователя
4. Создание учетной записи пользователя/учетная запись
5. Вход/выход из учетной записи

Нижняя часть шапки содержит ссылки на разделы, относящиеся к данной странице:
1. Cтраница
2. Обсуждение страницы
3. Чтение страницы
4. Просмотр кода страницы
5. История изменения страницы
6. Поле для поиска информации на сайте.

В левом блоке после логотипа сайта (кликабелен, ссылка на главную страницу) страницы идут сгруппированные по логическому признаку ссылки. Выравнивание по левой стороне.
Блоки:
1. Навигация - содержание, статьи, события + пожертвования
2. Участие - FAQ и ссылки на элементы сообщества
3. Инструменты для работы со страницей
4. Печать/экспорт
5. Информация по данной теме в других проектах экосистемы
6. Версия страницы на других языках
а также ссылка на редактирование.

В основном блоке представлена информация по предмету статьи в следующей структуре:
1. Краткое описание слева и карточка предмета с основными хараектеристиками справа
2. Содержание статьи
3. Текстовая и графическая информация, разбитая по параграфам. Графика или картинки располагаются в шахматном порядке.
4. Ссылки на связанные статьи/страницы
5. Комментарии к статье
6. Примечения
7. Список литературы 
8. Ссылки на классификаторы и словари

В подвале страницы находится информация о дате последнего редактирования статьи и лицензировании.
Также там расположены ссылки на юридическую информацию, контакты, статистику, разработчиков сайта.
Имеется ссылка перехода на мобильную версию.

Сайт в зависимости от масштаба на основаной странице может отображать одну или две информационных колонки.
Предусматривается стандарт оформления и форматирования текста (по левой стороне, с заголовками и подзаголовками, ссылками и т.д.)
